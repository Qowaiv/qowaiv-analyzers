# QW0027: Create UUID's using Uuid.Parse or Uuid.Empty
A [UUID](https://en.wikipedia.org/wiki/Universally_unique_identifier)  is a
128-bit number used to identify information in computer systems. UUID's can be
created by using `new Uuid(string)`, `Uuid.Parse(string)` and `Uuid.Empty`.

This rule generates a warning when the default constructor is used: If an
actual UUID was preferred, `Uuid.Parse()` should be used, otherwise `Uuid.Empty`.

## Non-compliant
``` C#
Uuid id = new();
```

## Compliant
``` C#
Uuid id = Uuid.Parse("1f570b45-8b18-496a-84a3-634db4f4c552");
```
or
``` C#
Uuid id = Uuid.Empty;
```
