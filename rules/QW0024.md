# QW0024: Create GUID's using Guid.Parse or Guid.Empty
A [GUID](https://en.wikipedia.org/wiki/Universally_unique_identifier)  is a
128-bit number used to identify information in computer systems. GUID's can be
created by using `new Guid(string)`, `Guid.Parse(string)` and `Guid.Empty`.

This rules generates a warning when the default constructor is used: If an
actual GUID was preferred, `Guid.Parse()` should be used, otherwise `Guid.Empty`.

## Non-compliant
``` C#
Guid id = new();
```

## Compliant
``` C#
Guid id = Guid.Parse("1f570b45-8b18-496a-84a3-634db4f4c552");
```
or
``` C#
Guid id = Guid.Empty;
```
