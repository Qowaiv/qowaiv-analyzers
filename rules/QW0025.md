# QW0022: Use compliant UUID literals
A [UUID](https://en.wikipedia.org/wiki/Universally_unique_identifier)  is a
128-bit number used to identify information in computer systems. There are
multiple usecases where a constant has to be created in code, wich can be done
by using `new Uuid(string)` and `Uuid.Parse(string)`.

This rule ensures that the provided string literal actually represents a valid UUID.

## Non-compliant
``` C#
var id = Uuid.Parse("no UUID");
```

## Compliant
``` C#
var id = Uuid.Parse("1f570b45-8b18-496a-84a3-634db4f4c552");
```
