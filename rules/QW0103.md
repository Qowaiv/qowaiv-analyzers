# QW0103: Decorate validation attributes

To ensure that [QW0102: Use compliant validation attributes](QW0102.md) can
report on invalid usage of validation attributes, validation attributes should
be decorated with the types they support, using the `[Validates]` attribute.

## Non-compliant
``` C#
[AttributeUsage(AttributeTargets.Property)]
public sealed class EvenNumberAttribute: ValidationAttribute
{
    public bool IsValid(object? value)
        => valus is int i && i % 2 == 0;
    }
}
```

## Compliant
``` C#
[AttributeUsage(AttributeTargets.Property)]
[Validates(typeof(int))]
public sealed class EvenNumberAttribute: ValidationAttribute
{
    public bool IsValid(object? value)
        => valus is int i && i % 2 == 0;
    }
}
```
