# QW0023: Provide GUID values
A [GUID](https://en.wikipedia.org/wiki/Universally_unique_identifier)  is a
128-bit number used to identify information in computer systems. There are
multiple usecases where a constant has to be created in code, wich can be done
by using `new Guid(string)` and `Guid.Parse(string)`.

This rule generates a warning when no argument is provided yet. By doing so,
The code fix can be applied to generate a valid GUID.

## Non-compliant
``` C#
var id = Guid.Parse();
```

## Compliant
``` C#
var id = Guid.Parse("1f570b45-8b18-496a-84a3-634db4f4c552");
```
