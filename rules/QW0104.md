# QW0104: Use validates attribute on validation attributes only

To ensure that [QW0102: Use compliant validation attributes](QW0102.md) can
report on invalid usage of validation attributes, validation attributes when
decorated with a `[Validates]` attribute. Decorating other types has meaning
and is most likely a mistake.

## Non-compliant
``` C#
[Validates(typeof(int))]
public sealed class EvenNumberAttribute : Attribute
{
}
```

## Compliant
``` C#
[Validates(typeof(int))]
public sealed class EvenNumberAttribute: ValidationAttribute
{
    public bool IsValid(object? value)
        => valus is int i && i % 2 == 0;
    }
}
```

or

``` C#
public sealed class EvenNumberAttribute : Attribute
{
}
```
