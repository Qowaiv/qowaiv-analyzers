# QW0019: Prefer XML to LINQ over DOM
Microsoft introduced [LINQ to XML](https://learn.microsoft.com/dotnet/standard/linq/linq-xml-overview)
with the release of [.NET 3.5](https://dotnet.microsoft.com) in 2008. With this
API, creation and manipulations of XML has dramatically improved compared to XmlDocument. On
top of that, memory usage and execution ties have been dramatically reduced.


## Non-compliant
``` C#
var doc = new XmlDocument();
doc.InnerXml = "<root><elm>42</elm></root>";
```

## Compliant
``` C#
var doc = XDocument.Parse("<root><elm>42</elm></root>");
````
