# QW0020:Prefer System.Text.Json over Newtonsoft.Json
Microsoft introduced [`System.Text.Json`](https://learn.microsoft.com/dotnet/standard/serialization/system-text-json/overview)
with the release of [.NET 5](https://dotnet.microsoft.com) in 2020. With .NET 6
- one year later - the features lacking in the first released where added.
Since then, there are hardly no scneario's left, where `Newtonsoft.Json` is
preferable over `System.Text.Json`. This rule advises to migrate to
`System.Text.Json`, which is:
- faster
- has a smaller memory footprint
- ships with  the .NET framework
