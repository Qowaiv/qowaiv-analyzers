# QW0021: Use leading zeros to define date constants
To improve the readability of date(time) literals it is preferred to comply to
the [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601) by adding leading zeros
to `year`, `month`, `day`, `hour`, `minutes`, `seconds`, and `milliseconds` if
needed.

## Non-compliant
``` C#
DateOnly date = new(2017, 6, 11);
```

## Compliant
``` C#
DateOnly date = new(2017, 06, 11);
```
